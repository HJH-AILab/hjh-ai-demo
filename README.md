# hjh-ai-demo

使用好机绘AIGC云服务示例，AI绘图主要玩法包括创意大变身、主角梦想秀

## 好机绘AIGC云服务接口文档

[接口文档](https://cloud.wepromo.cn/openapi/index.html)

## 主要接口

* 获取工作流
* AI绘图
* 查询AI绘图结果
